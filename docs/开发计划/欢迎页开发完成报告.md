## 💡 使用提示

### 重置引子动画
在浏览器控制台执行：
```javascript
localStorage.removeItem('construct_layer_intro_seen')
```
然后刷新页面即可重新观看引子动画。

### 测试继续游戏功能
在浏览器控制台执行：
```javascript
localStorage.setItem('construct_layer_save', 'test')
```
刷新页面后"继续游戏"按钮将被启用。
