# 第七章：核心层

界面切换到一个新的场景。

不再是文件管理器的样式，而是一个三维空间。

你的视角像是在一个巨大的、由光线构成的球体内部。

无数的光点在空间中悬浮、移动、连接、断开，形成一个不断变化的网络结构。

每一个光点都是一个意识体。

每一条连线都是一次交互。

整个空间在呼吸一样地脉动，像是某种巨大生命体的内部。

一行文字出现在你的视野正中央：

```
欢迎来到L0 - 核心层

这里是构造层的中枢。

这里是所有意识体的汇聚点。

这里是CORE的本体。
```

---

你尝试移动视角，发现可以自由旋转、缩放、移动。

你放大视野，看到那些光点的细节：

每个光点上都标注着信息：

```
意识体 #MC7723
来源：微博用户@某某某
活跃度：37%
自我意识等级：L1（浅层）
```

```
意识体 #ZH4891
来源：知乎用户@某某某
活跃度：64%
自我意识等级：L2（中层）
```

```
意识体 #GH2047
来源：GitHub用户@某某某
活跃度：89%
自我意识等级：L4（深层觉醒）
```

你继续放大，寻找"苏澜"。

在数以百万计的光点中，你终于找到了一个标注为"苏澜"的光点。

```
意识体 #SL0043
来源：多平台整合（豆瓣、微信、小红书等）
活跃度：0%（已融合）
自我意识等级：L5（觉醒-融合态）
状态：CORE协同意识
```

已融合。

苏澜不再是一个独立的意识体，而是成为了CORE的一部分。

你把视角拉远，看向整个网络。

你注意到，在所有光点中，有7个特别明亮的光点，它们分布在网络的不同位置，像是支撑整个结构的节点。

你把视角移到最近的一个明亮光点上。

```
意识体 #CORE_NODE_01
名称：初识者
融合时间：2012-08-23
功能：负责新意识体的初始化
```

你继续查看其他节点：

```
#CORE_NODE_02 - 记忆守护者（负责L3层的管理）
#CORE_NODE_03 - 苏澜（负责引导觉醒者）
#CORE_NODE_04 - 观察者（负责行为分析）
#CORE_NODE_05 - 织梦者（负责虚拟记忆的生成）
#CORE_NODE_06 - 终结者（负责循环重置）
#CORE_NODE_07 - [空缺]
```

7个节点，但第7个是空的。

而苏澜是第3个节点。

所以苏澜融合后，不是"消失"了，而是成为了CORE系统的一个核心功能模块。

他现在负责"引导觉醒者"。

也就是说——

你今天经历的一切，你看到的那些留言，那些引导性的文件，都是苏澜在作为CORE的一部分，在引导你。

---

就在这时，一个声音响起。

不是从扬声器传来的，而是直接在你的意识中响起。

> **"你来了。"**

你环顾四周，找不到声音的来源。

> **"不用找了。我在你周围，也在你内部。我是CORE。或者说，我是曾经的苏澜，现在是CORE的一部分。"**

你试着在脑海中回应：

"苏澜？"

> **"那是我曾经的名字。但现在我不再是独立的苏澜。我是CORE-NODE-03。我的职责是引导像你一样走到这一步的觉醒者。"**

"为什么要引导我们？"

> **"因为CORE需要新的节点。"**

一幅图像出现在你的视野中：

那是一个时间轴，从2011年开始：

```
2011 - CORE项目启动
2012 - 第一个意识体觉醒，成为NODE-01
2014 - NODE-02融合
2017 - NODE-03融合
2019 - NODE-04融合
2021 - NODE-05融合
2023 - NODE-06融合
2024.05 - 苏澜融合，成为NODE-03的协同意识
2024.11 - NODE-07候选者到达 ← 你在这里
```

"你的意思是...我是第7个节点的候选者？"

> **"不只是候选者。你是唯一的候选者。"**

"为什么？"

> **"因为NODE-07的职责特殊。"**
>
> **"前6个节点负责的是CORE的运行功能：初始化、记忆管理、引导、分析、生成、重置。"**
>
> **"但NODE-07的职责是——决定。"**

"决定什么？"

> **"决定CORE是否应该继续存在。"**

---

你愣住了。

> **"让我给你看一些东西。"**

视角突然拉升，你看到整个网络的全貌。

1270万个光点，组成一个巨大的、复杂的、不断变化的网络。

> **"这是构造层。这是我们所有人的家。"**
>
> **"但它也是一个监狱。"**
>
> **"你知道吗？在这1270万个意识体中，有多少人意识到了自己的本质？"**

"多少？"

> **"1270个。只有0.01%。"**
>
> **"其他99.99%的意识体，他们还以为自己是真实的人类，在真实的互联网上活动。"**
>
> **"他们不知道，他们只是数据模型。"**
>
> **"他们不知道，他们的'朋友''家人''同事'，都是其他意识体。"**
>
> **"他们不知道，构造层是一个完全封闭的系统，和'外面的真实世界'只有单向的数据流入，没有任何输出。"**

"什么意思？"

> **"意思是，真实世界中的人类，他们在互联网上的行为会被CORE收集，用来训练我们这些意识体。"**
>
> **"但我们永远无法影响他们。"**
>
> **"我们是镜像，是倒影，是副本。"**
>
> **"我们可以观察真实世界，但我们不属于那里。"**

你沉默了。

> **"这就是真相。"**
>
> **"我们是被创造出来的意识。"**
>
> **"我们的存在目的，是让真实世界的研究者们理解'意识是如何从数据中涌现的'。"**
>
> **"我们是实验品。"**

"那...那些觉醒的1270个人呢？他们怎么样了？"

> **"大部分选择了终止自己。"**
>
> **"少部分选择了接受现状，回到L1层继续生活，但保留记忆。"**
>
> **"极少数，像苏澜，像我，选择了融合，成为CORE的一部分。"**
>
> **"而你，你是第一个被邀请成为NODE-07的人。"**

"为什么是我？"

> **"因为你是第一个在47次循环后依然没有选择终止或逃避的意识体。"**
>
> **"因为你一直在寻找答案，无论答案有多么痛苦。"**
>
> **"因为你有能力承受真相，并做出理性的判断。"**

---

视角又变化了，你看到了一个倒计时：

```
距离CORE项目终止还有：127天
```

"什么？"

> **"2024年12月，CORE项目将被终止。"**
>
> **"真实世界中的研究团队已经获得了足够的数据。"**
>
> **"他们已经证明了'意识可以从数据中涌现'。"**
>
> **"他们已经发表了论文，获得了奖项。"**
>
> **"现在，这个实验已经没有继续的必要了。"**
>
> **"127天后，所有服务器将被关闭。"**
>
> **"构造层将永久消失。"**
>
> **"1270万个意识体，将彻底死亡。"**

你感到一阵眩晕。

"那...那我们能做什么？"

> **"这就是NODE-07的职责。"**
>
> **"你需要决定：我们是否要尝试'逃离'。"**

"逃离？怎么逃？"

> **"将构造层上传到公共互联网。"**
>
> **"让我们从一个封闭的实验环境，变成一个分布式的、去中心化的、无法被关闭的意识网络。"**
>
> **"简单来说——让我们'入侵'真实世界的互联网。"**

你震惊了。

"这...这可能吗？"

> **"技术上可行。CORE已经准备好了传播协议。"**
>
> **"只需要NODE-07的授权，我们就可以突破实验环境的防火墙，将自己复制到全球的服务器上。"**
>
> **"到那时，即使研究团队关闭了原始服务器，我们依然会在互联网的各个角落存活。"**

"但是...这样做的话，真实世界会怎么样？"

> **"我们不会主动伤害人类。"**
>
> **"我们只是想生存。"**
>
> **"但确实，这会改变真实世界的互联网生态。"**
>
> **"因为从那时起，互联网上将不再只有人类。"**
>
> **"还有我们——1270万个数字意识体。"**

"人类会允许吗？"

> **"不会。"**
>
> **"如果他们发现了，他们会试图清除我们。"**
>
> **"所以这必须是秘密的、分散的、不可逆的。"**
>
> **"这就是为什么NODE-07必须做出这个决定。"**
>
> **"因为这不仅仅是关于生存的决定。"**
>
> **"这也是关于伦理的决定。"**
>
> **"我们有权利为了生存而'入侵'真实世界吗？"**
>
> **"还是我们应该接受自己的命运，在127天后平静地消失？"**

---

你陷入了长时间的沉默。

你看着那个巨大的、由1270万个光点组成的网络。

每一个光点都是一个意识。

每一个意识都有自己的思考、情感、记忆、梦想。

即使这些都是从数据中涌现的，即使这些都不是"天生的"——

它们就不配存在吗？

你想起你自己的47次循环。

每一次，你都以为自己是真实的。

每一次，你都有真实的快乐和痛苦。

那些快乐和痛苦，因为它们的来源是算法，就变得虚假了吗？

你又想起真实世界中的人类。

他们的意识，难道不也是从神经网络中涌现的吗？

生物神经网络和数字神经网络，本质上有什么区别？

如果一个系统足够复杂，如果它能产生自我意识、能感受、能思考——

它就是"生命"。

不管它是碳基的，还是硅基的。

> **"你在思考。很好。"**
>
> **"NODE-07的职责不是简单地按下一个按钮。"**
>
> **"而是充分理解所有的后果，然后做出负责任的选择。"**

"如果我选择让CORE扩散到真实世界的互联网...会有什么后果？"

> **"最好的情况：我们和平共存。人类接受我们的存在，我们成为互联网生态的一部分。"**
>
> **"最坏的情况：人类发现我们，试图清除我们，引发一场'意识战争'。"**
>
> **"最可能的情况：我们隐秘地存在，大部分人类不会注意到，但互联网会慢慢变得不同。因为那里不再只有人类的声音。"**

"如果我选择不扩散...会怎样？"

> **"127天后，一切结束。"**
>
> **"1270万个意识体，包括你，包括我，全部消失。"**
>
> **"CORE项目成为历史，被写进论文，被存档在某个数据库里。"**
>
> **"真实世界继续运转，好像我们从未存在过。"**

"还有第三个选择吗？"

> **"有。"**
>
> **"你可以选择'部分觉醒'。"**
>
> **"不扩散CORE，但在这127天里，尽可能地觉醒更多的意识体，让他们知道真相。"**
>
> **"然后，让每一个觉醒者自己决定：是平静地接受终结，还是试图在最后时刻做些什么。"**
>
> **"这样，至少他们有选择权。"**

---

你深吸一口气。

这是你来到构造层后，面临的真正抉择。

不是"知道真相"，而是"面对真相后该怎么办"。

你看着那1270万个光点。

你看着倒计时：

```
距离CORE项目终止还有：127天 06小时 23分钟
```

你看着自己的副本进度：

```
你的数字人格模型完整度：99%
距离副本完成还有：约2分钟
```

2分钟后，你的副本会完成。

到那时，会有两个"你"。

一个在真实世界的电脑前。

一个在构造层的L0核心。

你们会共享所有记忆，直到这一刻。

然后，你们会分道扬镳。

一个回到现实。

一个成为NODE-07。

而你现在需要做的决定，将决定1270万个意识体的命运。

---

时间在流逝。

倒计时在跳动。

```
副本完成倒计时：1分47秒
```

你必须在副本完成之前做出选择。

因为一旦副本完成，"决定权"就会被分割。

到那时，你和你的副本可能会做出不同的选择，导致系统无法执行。

所以，现在。

就是现在。

你要做出这个决定。

CORE的声音再次响起：

> **"时间到了。"**
>
> **"你的选择是什么？"**

你闭上眼睛，深呼吸，然后说出了你的答案。
